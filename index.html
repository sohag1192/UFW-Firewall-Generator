What is UFW?

UFW (Uncomplicated Firewall) is a user-friendly frontend for iptables on Linux. It simplifies the process of configuring a firewall to protect your server.

Key Concepts:

  • Allow: Opens a port so traffic can come in (e.g., for a website).
  • Deny: Blocks traffic silently.
  • Reject: Blocks traffic but tells the sender "Connection refused".

Common Port Cheat Sheet:

22 : SSH (Remote Login)
80 : HTTP (Web Non-Secure)
443 : HTTPS (Web Secure)
21 : FTP (File Transfer)
3306 : MySQL Database
53 : DNS

Essential Commands:

  • Enable Firewall: sudo ufw enable
  • Check Status: sudo ufw status verbose
  • Reset Rules: sudo ufw reset
